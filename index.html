Search space:¶

Hierarqcal motifs (Compositions of Qcycle, Qpivot, Qmask, Qunmask)

  • are independent of the number of qubits
  • have well defined discrete variables (e.g. mapping, boundary,...)

Search strategy:¶

Discrete optimization algorithm

  • Random search
  • Genetic algorithm
  • Annealing

Performance estimate:¶

Fitness function

  • Fidelity w.r.t. some target states, $|\langle y^{(i)} | U |x^{(i)} \rangle|$
  • Classification performance score (accuracy, F1 score, etc.)
  • Expectation value (e.g. energy)

Penalties for number of

  • motifs
  • gates
  • parameters

Genetic algorithm¶

# Initialize the population, a set of evaluated motifs
population = randomly_initialize_population()
while fitness < threshold:
    # Select two members of the population to mutate
    genotype1, genotype2 = tournament_selection(population)
    # Generate new motifs
    children = (
        crossover(genotype1, genotype2), 
        mutation(genotype1), 
        mutation(genotype2)
    )
    # Evaluate the fitness of the new motifs
    for child in children:
        evaluate_genotypes(child)
    # Update the population
    population = update_population(children)

Quantum Fourier Transform¶


$ \sum_{j=0}^{2^n-1} z_j |j\rangle \rightarrow \frac{1}{\sqrt{2^n}} \sum^{N-1}_{k=0} \sum_{j=0}^{2^n-1} z_j e^{2\pi ijk/N}|k\rangle $


  • Found motif should work correctly for arbitrary circuit sizes
  • Repetitions (multiplication) should be learnt
  • Training set: 2, 3 and 4 qubits
  • Mappings include parameterized gates
